# ShoppingTest


### A shopping cart using the implementation of the MVC design Pattern
## Available Options: 

 - 1 ) Check products in store 
 - 2 ) Find a product
 - 3 ) Buy a product 
 - 4 ) Check total price per item in cart 
 - 5 ) Get cart details
 - 6 ) Remaining items in store
 - 0 ) Quit
